## 📝 Описание

**Genetic Algorithm Optimization** — это десктопное приложение с графическим интерфейсом, реализующее генетический алгоритм для решения задач безусловной оптимизации. Программа позволяет находить глобальный минимум или максимум функции двух переменных методом эволюционного поиска.

Приложение разработано на языке Python с использованием библиотек Tkinter (графический интерфейс), Matplotlib (визуализация) и NumPy (математические вычисления).

### Что такое генетический алгоритм?

Генетический алгоритм — это метод оптимизации, основанный на механизмах естественного отбора и наследственности:
- **Популяция** — множество потенциальных решений
- **Селекция** — отбор лучших особей
- **Кроссинговер** — скрещивание для создания потомков
- **Мутация** — случайные изменения для разнообразия
- **Элитизм** — сохранение лучших решений

⚙️ Параметры алгоритма
Параметр	Описание	Рекомендуемое значение
Размер популяции	Количество особей в поколении	50–200
Количество поколений	Число итераций алгоритма	30–200
Вероятность мутации	Частота случайных изменений	0.01–0.1
Вероятность скрещивания	Частота обмена генами	0.6–0.9
Размер элиты	Лучшие особи, переходящие в следующее поколение	1–5
Влияние параметров
Большая популяция → лучше исследует пространство, но медленнее

Высокая мутация → больше разнообразия, но может потерять решение

Низкое скрещивание → медленный обмен информацией

Большая элита → стабильнее, но меньше разнообразия
🧠 Как это работает
Алгоритм по шагам
text
1. ИНИЦИАЛИЗАЦИЯ → Создание случайной популяции (N точек)
2. ОЦЕНКА → Вычисление fitness для каждой точки
3. ПРОВЕРКА ОСТАНОВА → Если достигнуто max поколений → выход
4. СЕЛЕКЦИЯ → Выбор лучших родителей (турнирный отбор)
5. КРОССИНГОВЕР → Создание потомков C = α·P₁ + (1-α)·P₂
6. МУТАЦИЯ → Случайное изменение x' = x + N(0, σ)
7. ЭЛИТИЗМ → Сохранение лучших особей
8. НОВОЕ ПОКОЛЕНИЕ → Замена популяции, переход к шагу 2
Математические формулы
Кроссинговер:

text
C₁ = α·P₁ + (1-α)·P₂
C₂ = (1-α)·P₁ + α·P₂
Мутация:

text
x' = x + N(0, 0.5)
Функция приспособленности:

text
fitness = f(x, y)           # для минимума
fitness = -f(x, y)          # для максимума (преобразование)
